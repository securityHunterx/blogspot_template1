Customize Template

Top Area

Header Background

Go to Blogger.com -> Your Blog -> Template -> Edit HTML And Find ( CTRL + F ) This:- <!-- Header Background -->
Now Change Header Background With Yours

Header Text

Go to Blogger.com -> Your Blog -> Template -> Edit HTML And Find ( CTRL + F ) This:- <!-- ====== Header Text====== -->

Now Change Slides Text With Yours

Menu Navigation Items

Go to Blogger.com -> Your Blog -> Template -> Edit HTML And Find ( CTRL + F ) This:- <!-- Top Menu -->

Now Change It With Yours

Header Area

Logo Setting:

Go to your Blogger Account - Your Blog & select Layout

Now In Header Widget Click On Edit

Upload Your Logo And Select "Instead of title and description" as the "Placement" option & Select Shrink To Fit.

About

Go to Blogger.com -> Your Blog -> Template -> Edit HTML And One By One Find ( CTRL + F ) All This:-

  1. <!-- About Section Heading --> (Add Your Title)
  2. <!-- About Section Sub Heading --> (Add Your Sub Heading)
  3. <!-- About Section Content --> (Add Your Content)
  4. <!-- About Section Social Button --> (Add Your Link)

Sections

Go to Blogger.com -> Your Blog -> Template -> Edit HTML And One By One Find ( CTRL + F ) All This:-

  1. Find :- <!-- Our Skills Section -->
    1. [In Skills Section] Change The
    2. The Title (Find <!-- Our Skills Title -->)
    3. Label (Find <!-- Our Skills Label -->)
  2. Find :- <!-- Services Section -->
    1. [In Services Section] Change The:-
    2. Icon (Find <!-- Services Section Icon -->)
    3. Title (Find <!-- Services Section Title -->)
    4. Content (Find <!-- Services Section Content -->)
  3. Find :- <!-- Contact Us Section --> (Note:- The Contact Form Work Automatically)
    1. [In CONTACT Section] Change The:-
    2. Details (Find <!-- Contact Section Details -->)
  4. Find :- <!-- Footer -->
    1. [In Footer Section] Change The:-
    2. Social Links (Find <!-- Footer Social Links-->)

Settings

Mobile Version

To Use Responsive Design In Mobile Devices Than First You Need To Enable It.

Note:The Responsive Menu Link Sub Menu Will Not Work(Open) In The Desktop If You Resize Your Window It Is Only Work On Mobile Devices.

  1. Go to Blogger.com -> Your Blog -> Template -> Now In Mobile Section Click On Gear Button As Shown In Given Below Image

2. Click On "No. Show desktop template on mobile devices." And Than Click On Save.

Change The Post Date Format:

Go to Blogger Dashboard -> Settings -> Language and Formatting -> Formatting -> Timestamp Format -> Set It To Date Instead Of Time.
Now Click On Save...
